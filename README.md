# Unemployment---Analysis---Python-
Unemployment Analysis with Python

## Introduction
This project analyzes unemployment trends in India using Python.

## Dataset
Dataset Used: Unemployment in India.csv

## Tools and Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Objectives
- Analyze unemployment rates in India.
- Study state-wise unemployment trends.
- Investigate the impact of Covid-19 on unemployment.
- Visualize data using charts and graphs.

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- State-wise Analysis
- Rural vs Urban Analysis
- Covid-19 Impact Analysis
- Data Visualization

## Conclusion
The analysis shows that unemployment rates vary across different states of India. A significant increase in unemployment was observed during the Covid-19 period. Data visualization helped identify important trends and patterns in the dataset.
